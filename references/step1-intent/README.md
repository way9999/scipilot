# Step 1: 意图对齐 (Intent Alignment)

> 通过尽量少且高信息密度的问题，引导模型快速识别用户的研究意图、学科背景与开题报告的核心方向。

## 文件清单

| 文件 | 用途 |
|------|------|
| [questions-mvp.md](questions-mvp.md) | MVP模式问题库（5问） |
| [questions-full.md](questions-full.md) | Full模式问题库（12问） |
| [domain-adapters.md](domain-adapters.md) | 学科领域适配器（Q4/Q5选项适配） |

## 输出

- 用户研究意图的结构化记录
- 学科方向确定（供后续Step使用）
