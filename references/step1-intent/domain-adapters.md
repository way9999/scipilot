# 领域适配器（Domain Adapters）

> 根据 Q2（学科方向）自动选择对应的选项集

## 使用方式

1. Q2 确定学科后，匹配下方对应领域
2. Q4、Q5 使用该领域的选项替换默认选项
3. 若学科未列出，使用"通用"选项或请用户自述

## 适配流程伪代码

```
function adaptQuestions(discipline):
    if discipline in DOMAIN_MAP:
        return DOMAIN_MAP[discipline]
    else if partialMatch(discipline):
        return nearestDomain(discipline)
    else:
        return GENERIC_TEMPLATE
```
