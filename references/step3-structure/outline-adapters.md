# 学科大纲适配器 (Outline Adapters)

> 根据 Q2（学科方向）自动选择对应的大纲模板与篇幅配置

## 使用方式

1. Step 1 完成后，根据 Q2 匹配下方对应学科
2. 加载该学科的大纲模板与篇幅建议
3. 若学科未列出，使用"通用"模板

## 适配函数

```python
def get_outline_template(discipline):
    ADAPTERS = {
        "计算机": CS_TEMPLATE,
        "人工智能": CS_TEMPLATE,
        "材料": MATERIALS_TEMPLATE,
        "生物": BIO_TEMPLATE,
        "能动": ENERGY_TEMPLATE,
        "物理": PHYSICS_TEMPLATE,
        "化学": CHEMISTRY_TEMPLATE,
        "经济": ECON_TEMPLATE,
        "管理": ECON_TEMPLATE,
    }

    for key, template in ADAPTERS.items():
        if key in discipline:
            return template

    return GENERIC_TEMPLATE
```
